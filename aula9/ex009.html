Aprendendo a utilizar o DOM em Javascript

A árvore DOM constitui uma cadeia de elementos hierárquicos que podem ser acessados à partir do elemento 'window' (o pai de todos).
Esses elementos podem ser acessados por:
Marca - São as tags ou elementos nativos da linguagem HTML utilizando '.getElementsByTagName()' em 'document'
ID - São identidades geradas nos elementos HTML como na instrução 'div id="id"' e utilizando '.getElementById()'
Nome - Da mesma forma que no elemento ID 'div name="nome"' utilizando 'getElementsByName()'
Classe - Também segue a mesma regra 'div class="classe"' utilizando 'getElementsByClassName()'
Seletor - Destina pelo seletor CSS utilizando 'querySelector()' ou 'querySelectorAll()'
